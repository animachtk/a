Ранги:
Админ сайта - красный
Админ канала - фиолетовый, синий
Модератор канала - зелёный
Пользователь - белый
Гость - серый
Добавление видео:
Видео можно добавлять с:
YouTube,
Twitch,
sibnet,
и по некоторым прямым ссылкам (без привязки по ip)

Основные функции:
- пропустить(скипнуть) текущее видео(Недоступно во время трансляций по голосованию.)
- открыть панель смайлов канала.
- открыть панель избранных картинок.
- открыть панель шрифтов.
- загрузить свою картинку.
- Добавить своё видео.
