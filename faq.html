Добавление видео:
Видео можно добавлять с:
YouTube,
Twitch,
sibnet,
и по некоторым прямым ссылкам (без привязки по ip)

Основные функции:
- пропустить(скипнуть) текущее видео(Недоступно во время трансляций по голосованию.)
- открыть панель смайлов канала.
- открыть панель избранных картинок.
- открыть панель шрифтов.
- загрузить свою картинку.
- Добавить своё видео.
Добавление видео
Видео добавляется со следующих сайтов:
www.myvi.tv - при добавлении недавно добавленных на данный ресурс видео может возникнуть проблема. Если вы скопировали URL видео из плеера, но при добавлении выскакивает красное окно об ошибке, то необходимо при нажатии ПКМ выбрать пункт "Открыть видео в новой вкладке" и скопировать ссылку из адресной строки браузера в новой вкладке.

sibnet.ru - Под плеером будет иметься кнопка . Её необходимо нажать и тогда появится строка с выбором - необходимо выбрать "Полностью". После этого жмём ПКМ по видео и выбираем "Открыть видео в новой вкладке". И копируем ссылку по аналогии с myvi.tv, т.е. из адресной строки браузера в новой вкладке (в правильной ссылке на конце должно содержаться "noip=1"). Внимание: ссылки с сибнета работают в течении нескольких часов, после чего они перестают открываться и необходимо получать новую ссылку.

smotretanime.ru - здесь достаточно просто кликнуть ПКМ и "скопировать URL видео". Но при добавлении видео с сайта smotretanime.ru и некоторых других в строке, куда нужно вставить текст с адресом ссылки, необходимо ввести перед ссылкой "fi:", в ином случае видео не добавится и появится красное сообщение об ошибке. "fi:" также вызывает строку для ввода названия видео.

animevost.org - На сайте имеется кнопка скачивания. Просто кликаем по ней ПКМ и нажимаем "скопировать URL (ссылку)". Извлечённые ссылки не всегда годятся для добавления, но попробовать стоит если нет других вариантов.

Базы видео с аниме:
findanime.org - Содержит в себе плееры с сайтов sibnet и myvi. Также есть плеер с сайта kiwi, из которого тоже можно извлечь ссылку и добавить (но скорость низкая и видео будет зависать).
animeonline.su - Собственный плеер (myvi) из которого легко добавлять. Если просит установить Flash плеер, то данное видео не добавить и ищем другой сайт.
animespirit.ru - Содержит в себе плееры с сайтов sibnet и myvi (в основном старый сайт myvi который заблокирован РосКомНадзором. Поэтому можно добавлять только новые тайтлы 2018 года начиная с осеннего сезона). -Перезалив старых тайтлов!
seasonvar.ru - Ищем ссылку в коде страницы открыв консоль разработчика (клавиша F12). При открытом окне консоли разработчика нажимаем сочетание клавиш "CTRL + F" и вводим в поиске ".mp4" затем в найденных результатах ищем значение похожее на обычную ссылку с .mp4" на конце и пробуем добавить. Либо есть ещё более легкий способ (см. общие правила по добавлению способ 2 ниже).

Общие правила по добавлению.
Чтобы добавить видео в плейлист нужно нажать кнопку и вставить ссылку. Для добавления названия необходимо вызвать ещё одну строку. Чтобы это сделать необходимо в момент когда курсор активен в строке с добавлением видео нажать кнопку "вниз" на клавиатуре. Тогда снизу появится ещё одна строка в которую нужно будет ввести название.
Способ 1. На большинстве сайтов достаточно просто кликнуть по видео правой кнопкой мыши и нажать "Скопировать URL".
Способ 2. Для удобства можно скачать расширение для браузера, которое будет обнаруживать ссылки автоматически и их можно будет легко скопировать. Например, для браузера хром можно скачать расширение . Данный аддон будет полезен для быстрого извлечения ссылок с сайта seasonvar.ru и некоторых других.
